Instructions:

  1. Open this file in your browser
  2. See which method displays the image correctly
  3. If none work, you need to check the sharing settings

To fix sharing:

  1. Go to Google Drive
  2. Right-click "SF Logo Transparent.jpg"
  3. Click "Share"
  4. Click "Change to anyone with the link"
  5. Make sure it says "Anyone with the link can view"
  6. Click "Copy link" and "Done"
